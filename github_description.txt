🚀 Job Board Platform - منصة وظائف احترافية

نظام متكامل لإدارة الوظائف والتقديمات مع نظام المقالات والإعجابات. مبني بـ Laravel 12.34.0 و Tailwind CSS.

✨ المميزات:
📝 نظام المقالات مع الإعجابات
💼 إدارة التقديمات المتقدمة
👥 صلاحيات متعددة (Admin, Company, User)
🎨 واجهة مستخدم احترافية
🔐 أمان وحماية متقدمة

🛠️ التقنيات: Laravel 12.34.0, PHP 8.2, MySQL, Tailwind CSS
